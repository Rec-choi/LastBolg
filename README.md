<h1 align="center">个人博客后台管理系统</h1>

<p align="center"><b>基于</b>Fantastic-admin框架</p>


## 特点

- 丰富的布局与主题，覆盖市面上各种中后台应用场景，兼容PC、平板和移动端
- 提供系统配置文件，轻松实现个性化定制
- 精心设计的动效，让每一处的动画都干净利落
- 根据路由配置自动生成导航栏
- 基于文件系统的路由
- 支持全方位权限验证
- 内置多级路由最佳缓存方案
- 轻松实现国际化多语言适配
- 提供接近于浏览器原生标签栏操作体验的标签页功能
